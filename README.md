# 🚀 PC Doctor

An advanced, lightweight, and all-in-one system optimization and cleaning utility custom-built for Windows. Optimized fully for performance and legacy system support (including Windows 7).

---

## 🛠️ Features / Özellikler

### 🌍 Multi-Language Support / Çoklu Dil Desteği
- Full integration with **English (EN)** and **Turkish (TR)** interfaces.

### 📊 Real-Time Monitoring / Anlık Takip
- **CPU & RAM:** Live performance tracking bars.
- **Storage:** Dynamic monitoring for all connected drives.
- **Top 3 Heavy Apps:** Instantly lists the top 3 background applications consuming your system resources.
- **On-Screen Display (OSD):** Toggleable overlay to track CPU/RAM status globally while gaming or working.

### ⚡ Optimization & Cleaning / Sistem Temizliği
- **One-Click Smart Clean:** Flushes DNS cache, purges temporary junk logs (`Temp` & `Windows\Temp`), and stabilizes system resource consumption instantly.
- **Process Killer:** Force-terminate heavy-resource background tasks directly from the UI with a single click (`X`).
